5.10 Random numbers

Notes

Intro

Random numbers are useful on software development for many reasons - game mecahanics, generating usernames/passwords, anytime you need something seemingly unpredictible.

Pseudo-random numbers - computers are just big calculators solving formulas and can't create truly random numbers, but insead, use an algorithm that generates that seems random

Random numbers in JS

Math.random() is a function that returns a pseudo-random number between 0 and 1, not including 1

Math.floor() is a function that takes a decimal number and rounds it down - chops off the decimal part

Random number calculations

To produce a random integer from a min value to a max value (including the min max) use this calculation

Math.floor(Math.random() * (max + 1 )) + min

Example

Generate a random number from 1 to 10 adn store it in a variable

var randnum = Math.floor(Math,random() * 11) + 1;
